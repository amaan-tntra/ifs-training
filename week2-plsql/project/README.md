# Week 2 – PL/SQL & DevOps Basics

This week focuses on learning advanced PL/SQL concepts and applying them through a structured practice project.

---

## Topics Covered

### PL/SQL Concepts
- Procedures
- Functions
- Cursors
- Triggers

### DevOps Learning
- Introduction to GitHub / GitLab
- Basic Git commands:
  - commit
  - push
  - pull
  - branching

---

## PL/SQL Practice

The `plsql/` folder contains individual scripts for learning:

- `procedures.sql` → procedure implementations  
- `functions.sql` → function examples  
- `cursors.sql` → cursor usage  
- `triggers.sql` → trigger implementation  

---

##  Practice Project

### Department Analytics System

A mini project built using PL/SQL concepts to analyze employee and department data.

### Features:
- Calculate average salary per department  
- Identify highest salary in each department  
- Count employees per department  
- Display employee details using cursors  
- Validate salary using triggers  

---

## Learning Outcome

- Understand and implement PL/SQL concepts  
- Write reusable database logic using procedures and functions  
- Work with cursors for data processing  
- Use triggers for data validation  
- Apply Git for version control
