# Azure Basics

## What is Azure

- Azure is Microsoft's cloud computing platform.
- It provides services for building, deploying, and managing applications.

---

## Common Azure Services

### Virtual Machines
- Cloud-hosted virtual servers

### Blob Storage
- Object/file storage service

### Azure SQL Database
- Managed SQL database service

### Virtual Network
- Networking service for cloud resources

### Azure Active Directory
- Identity and access management

---

## Typical Azure Architecture

```Users → App Service / VM → Azure SQL Database```

--- 

### Azure Use Cases
- Enterprise application hosting
- Database services
- Identity management
- Cloud infrastructure management

---

## Summary
- Azure provides enterprise-grade cloud services and infrastructure
- Commonly used in organizations using Microsoft ecosystem
