# Cloud Platform Fundamentals

## What is Cloud Computing

- Cloud computing is the delivery of computing services over the internet.
- It provides on-demand access to:
  - Servers
  - Storage
  - Databases
  - Networking
  - Software

---

## Benefits of Cloud

- Scalability
- High availability
- Cost efficiency
- Reduced infrastructure management
- Faster deployment

---

## Cloud Service Models

### Infrastructure as a Service (IaaS)
- Provides virtualized infrastructure
- Example:
  - Virtual Machines
  - Storage

### Platform as a Service (PaaS)
- Provides platform for application development
- Example:
  - Managed databases
  - App hosting

### Software as a Service (SaaS)
- Provides ready-to-use software over internet
- Example:
  - Gmail
  - Microsoft 365

---

## Deployment Models

- Public Cloud
- Private Cloud
- Hybrid Cloud

---

## Common Cloud Components

- Compute
- Storage

- Databases

## Cloud in Enterprise Systems

- Host ERP applications
- Run databases
- Enable disaster recovery
- Improve scalability

---

## Summary

- Cloud platforms provide scalable infrastructure and services over the internet
- They reduce operational overhead and improve deployment flexibility
