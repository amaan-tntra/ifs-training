# AWS Basics

## What is AWS

- AWS (Amazon Web Services) is a cloud computing platform by Amazon.
- It provides infrastructure and cloud services for hosting applications and systems.

---

## Common AWS Services

### EC2
- Virtual servers in the cloud

### S3
- Object storage service

### RDS
- Managed relational database service

### VPC
- Virtual private network environment

### IAM
- Identity and access management

---

## Typical AWS Architecture

```Users → Load Balancer → EC2 → RDS```

--- 

### AWS Use Cases
- Application hosting
- Database hosting
- Backup and storage
- Disaster recovery

---

## Summary
- AWS provides scalable and reliable cloud infrastructure
- Widely used for enterprise application hosting
